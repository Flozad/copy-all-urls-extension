# Context Menu

Access copy and paste functions directly from the right-click menu on any webpage.

## Using Context Menu

### Copy URLs
1. Right-click anywhere on a webpage
2. Select "Copy URLs"
3. All URLs from open tabs are copied to your clipboard

### Paste URLs
1. Right-click anywhere on a webpage
2. Select "Paste URLs"
3. URLs from your clipboard open in new tabs

## Context Menu Features

- **Always available**: Works from any webpage
- **Quick access**: No need to click the extension icon
- **Consistent behavior**: Uses your default format settings
- **Smart detection**: Automatically extracts URLs from various formats

## Enabling/Disabling

By default, context menu items are enabled. To hide them:

1. Open the Umbrella popup
2. Click **Options**
3. Find "Show context menu items"
4. Uncheck the box to hide context menu
5. Click **Save Settings**

### Why Disable Context Menu?

You might want to disable the context menu if:

- You prefer using keyboard shortcuts
- You want a cleaner right-click menu
- You rarely use the copy/paste functions
- Other extensions already provide similar functionality

## When to Use Context Menu

The context menu is ideal when:

- **Working with forms**: Quick access without switching focus
- **Multi-window workflows**: Copy tabs from different windows
- **No keyboard access**: Using a tablet or touch device
- **Context switching**: Moving between different tasks

## Troubleshooting

If context menu items don't appear:

1. Check that "Show context menu items" is enabled in Options
2. Refresh the webpage after changing settings
3. Check if another extension is conflicting
4. Restart Chrome if the issue persists
